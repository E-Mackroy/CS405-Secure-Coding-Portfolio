# CS405-Secure-Coding-Portfolio
Portfolio submission for SNHU CS-405 Secure Coding course


This repository contains my portfolio work for the CS-405 Secure Coding course at Southern New Hampshire University. It includes the major projects and assignments completed during the term, along with documentation that reflects what I learned and how I applied secure coding practices in each project.

#Included Files
#Project One – Security Policy

A written policy that covers secure coding guidelines, risks, and recommendations for a fictional organization. This project demonstrates my ability to outline clear practices that support safe software development.

#Project Two – Presentation

A presentation that explains how secure coding concepts were applied throughout the course. It includes examples of vulnerabilities, remediation strategies, and lessons learned.

#Coding Project ZIP Files

These ZIP files include my coding assignments from the course. Each project shows how secure coding principles were used to address specific problems, such as buffer overflows, input validation, and error handling.

#Portfolio Journal Reflection

A written reflection that covers what I learned in CS-405, how my skills have grown, and how I plan to use these skills in future development work.

#How to View the Work

Each file can be downloaded directly from this repository.
PowerPoint, Word documents, and ZIP files can be opened locally after downloading.
